# mvmorten.com

Source for [mvmorten.com](https://mvmorten.com) — the personal portfolio of Michael Mortensen.

A static, single-page site: plain HTML, custom CSS, and a small bit of vanilla JS. No framework, no build step.

## Structure

- `index.html` — page markup
- `css/styles.css` — stylesheet
- `js/scripts.js` — sticky nav, mobile menu, scroll reveal
- `assets/img/` — project illustrations and favicon

## Run locally

Open `index.html` in a browser. For a quick local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
